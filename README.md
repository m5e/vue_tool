# vue_tool

## Set up
下記コマンド実行
```
npm install
```
## Start up
1. 下記2つのコマンドを実行して起動
(それぞれ別ターミナルで実行)
* ビルドが成功すると、アクセス先のパスが表示される
```
npm run st
```
* コンソール上に`Listening on port 3000`というメッセージが表示されればOK
```
npm run se
```

2. 起動成功時にコンソール上に表示されるパスにアクセスし、つながればOK  
(8080が使用されてたらおそらく8081)

### When building
```
npm run bu
```

### Apply formatting to files
```
npm run li
```
