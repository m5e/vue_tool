# vue_tool

## セットアップ
1. 下記コマンド実行
```
npm install
```
2. 下記2つのコマンドを実行して起動
(それぞれ別ターミナルで実行)
* ビルドが成功すると、アクセス先のパスが表示される
```
npm run st
```
* コンソール上に`Listening on port 3000`というメッセージが表示されればOK
```
npm run se
```

3. localhost8080にアクセスしてつながればOK  
(8080が使用されてたらおそらく8081)

### ビルドするとき
```
npm run bu
```

### ファイルにフォーマット効かせたいとき
```
npm run li
```
